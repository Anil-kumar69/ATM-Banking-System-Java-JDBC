Project Overview:
Project Title: ATM Banking System (Java Console Application)
Description: A fully functional ATM simulation system with user and admin modules, built using Core Java, JDBC, MySQL, and OOP principles. The system supports secure banking operations, transaction logging, role-based access, and database persistence.

Key Features:
Admin Module: Create, view, and delete bank accounts

User Module: Withdraw, deposit, check balance, view transaction history

Secure Authentication: PIN-based login for users and admin

Database Integration: MySQL + JDBC for persistent storage

Transaction Logging: All financial transactions are recorded with timestamps

Console UI: Color-coded console interface for better user experience

Custom Exception Handling: Invalid input and business rule validations

Tech Stack:
Java (Core, OOP, Collections, Exception Handling)

JDBC (Database Connectivity)

MySQL (Database)

Scanner Class (Console I/O)
